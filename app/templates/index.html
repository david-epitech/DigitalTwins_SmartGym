Model Summary

    Why LSTM

    Athlete state evolves over time. Injury risk depends on recent time steps, not a single instant, so an LSTM is suitable for sequential workout signals.

    Input Features

      Output

      The model predicts an injury risk score from 0 to 100.

      Evaluation Metrics

      Low MAE means average prediction error is small. High R2 means the model explains a large part of target variance.

      Saved Model File

      models/smart_gym_lstm.keras stores the trained architecture and weights, so the backend can reuse the model directly without retraining.

      Digital Twin Explanation

      This project is a Digital Twin because it represents an athlete with a virtual monitoring layer. Workout variables simulate real-time state updates, and the AI model predicts risk states to support faster coaching decisions.
